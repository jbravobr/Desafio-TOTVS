# Desafio TOTOVS - TO BRASIL BACKEND

Desafio proposto para desenvolvedor Backend .NET 

## Pré-requisitos

Para executar corretamente esta aplicação é necessário a completa e atualizada instalação do Visual Studio 2019 e instalação do Docker (container Linux)

Esta aplicacão utiliza as seguintes portas HTTPS: **27017**, **8081** e **443**

## Instalação

Descompactar o arquivo ZIP. Dentro do conteúdo descompactado abrir o arquivo **Desafio-TOTOVS.sln** usando a versão mais recente do Visual Studio. Usar como projeto Inicial o projeto **docker-compose**
